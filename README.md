# Winn. ✦

> Célèbre chaque victoire. Transforme chaque épreuve.

## Déployer sur Vercel

### 1. Installer les dépendances
```bash
npm install
```

### 2. Tester en local
```bash
npm run dev
```

### 3. Déployer sur Vercel
- Va sur [vercel.com](https://vercel.com)
- Clique **"Add New Project"**
- Importe ce dépôt GitHub
- Clique **"Deploy"** — c'est tout !

## Stack
- React 18
- Vite
- Claude API (transformation philosophique + portrait IA)
